# Pretrained Weights

Download the following pretrained backbone weights and place them in `weights/pkl/`:

## Swin-Base (ImageNet-21K)

Used by BEACON and Mask2Former baseline configs.

Download `model_final_83d103.pkl` from the [Mask2Former Model Zoo](https://github.com/facebookresearch/Mask2Former/blob/main/MODEL_ZOO.md) (Swin-B, COCO Instance Segmentation, 50 epochs).

Place as: `weights/pkl/model_final_83d103.pkl`

## ResNet-50

Used by Mask R-CNN baseline configs.

Download from [Detectron2 Model Zoo](https://github.com/facebookresearch/detectron2/blob/main/MODEL_ZOO.md).

Place as: `weights/pkl/resnet-50.pkl`
